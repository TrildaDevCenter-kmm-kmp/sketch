GenericViewTarget

abstract class GenericViewTarget<T : View>(view: T) : ViewTarget<T> , TransitionViewTarget, LifecycleEventObserver, AttachObserver

An opinionated ViewTarget that simplifies updating the Image attached to a View and supports automatically starting and stopping animated Drawables.

If you need custom behaviour that this class doesn't support it's recommended to implement ViewTarget directly.
